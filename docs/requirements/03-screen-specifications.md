# 03 - 画面仕様

## 画面一覧

| 画面名 | ルート | 説明 |
|--------|--------|------|
| オンボーディング | (モーダル) | 初回起動時のみ表示。モード紹介 |
| メモ一覧 | `app/index.tsx` | メイン画面。メモのリスト表示 |
| メモ編集 | `app/memo/[id].tsx` | メモのテキスト編集 |

---

## 1. オンボーディング画面

### レイアウト

```
┌─────────────────────────┐
│                 [スキップ]│
│                          │
│    ┌──────────────────┐  │
│    │                  │  │
│    │   モードの        │  │
│    │   ビジュアル      │  │
│    │   プレビュー      │  │
│    │                  │  │
│    └──────────────────┘  │
│                          │
│    Fountain Pen          │
│    万年筆のような        │
│    書き心地を体験         │
│                          │
│    ● ○ ○                │
│                          │
│    [ はじめる ]           │
│                          │
└─────────────────────────┘
```

### 要素と振る舞い

| 要素 | 仕様 |
|------|------|
| ページ数 | 1〜2ページ（スワイプでページ送り） |
| ページ1 | 3つのモードの概要紹介 |
| ページ2（オプション） | 操作説明（スワイプ削除、長押し並び替え） |
| スキップボタン | 右上に配置。タップで即座にメモ一覧へ |
| はじめるボタン | 最終ページにのみ表示。タップでメモ一覧へ |
| ページインジケーター | 下部にドットインジケーター |
| 再表示 | なし（AsyncStorageでフラグ管理） |

## 2. メモ一覧画面（`app/index.tsx`）

### レイアウト

```
┌─────────────────────────┐
│  InkFlow    [🖋] [⌨] [✒]│  ← ヘッダー（モード切替アイコン）
├─────────────────────────┤
│                          │
│  ☐ 牛乳を買う      12:30 │  ← メモアイテム
│  ─────────────────────── │
│  ☑ 企画書を出す    昨日   │  ← 完了メモ（取り消し線）
│  ─────────────────────── │
│  ☐ 部屋を掃除する  3/1   │
│  ─────────────────────── │
│                          │
│                          │
│                          │
│                          │
│                     [＋] │  ← FAB（新規作成）
└─────────────────────────┘
```

### ヘッダー

| 要素 | 仕様 |
|------|------|
| アプリタイトル | 「InkFlow」を選択中モードのフォントで表示 |
| モード切替アイコン | 3つのアイコン（🖋 ⌨ ✒）を横並びで配置 |
| 選択中のアイコン | アクセントカラーでハイライト |
| 非選択のアイコン | 薄いグレー表示 |

### メモリスト

| 要素 | 仕様 |
|------|------|
| リスト実装 | react-native-draggable-flatlist |
| 各アイテム | チェックボックス + テキスト + 更新日時 |
| チェックボックス | 左端に配置。タップで完了/未完了切替 |
| テキスト | モードに応じたフォントで表示。1行、はみ出し時は省略（...） |
| 更新日時 | 右端に小さく表示。「12:30」「昨日」「3/1」の相対/短縮形式 |
| タップ | メモ編集画面へ遷移 |
| 左スワイプ | メモ削除（Undoトースト表示） |
| 長押し | ドラッグモード開始 |
| セパレーター | モードに応じたカラーの細い線 |

### 日時表示フォーマット

| 条件 | 表示例 |
|------|--------|
| 今日 | `12:30`（時刻のみ） |
| 昨日 | `昨日` |
| 今年内 | `3/1`（月/日） |
| 去年以前 | `2025/3/1`（年/月/日） |

### FAB（Floating Action Button）

| 要素 | 仕様 |
|------|------|
| 位置 | 画面右下（SafeArea内） |
| デザイン | 円形、モードのアクセントカラー、＋アイコン |
| タップ | 新規メモを作成し、編集画面へ遷移 |
| 上限到達時 | 非活性（グレーアウト）、タップ時にメッセージ表示 |

### 空状態（EmptyState）

```
┌─────────────────────────┐
│  InkFlow    [🖋] [⌨] [✒]│
├─────────────────────────┤
│                          │
│                          │
│                          │
│         ✒️               │
│    はじめてのメモを       │
│    書いてみましょう       │
│                          │
│                          │
│                          │
│                     [＋] │
└─────────────────────────┘
```

| 要素 | 仕様 |
|------|------|
| アイコン | モードに応じた筆記具アイコン |
| テキスト | 「はじめてのメモを書いてみましょう」 |
| フォント | モードに応じたフォント |

### Undoトースト

| 要素 | 仕様 |
|------|------|
| 位置 | 画面下部（FABの上） |
| 表示時間 | 3秒（タップで即座に消える） |
| テキスト | 「メモを削除しました」 |
| ボタン | 「元に戻す」 |
| アニメーション | 下からスライドイン、フェードアウト |

## 3. メモ編集画面（`app/memo/[id].tsx`）

### レイアウト

```
┌─────────────────────────┐
│  [←]            95/100   │  ← ヘッダー（戻る + 文字数）
├─────────────────────────┤
│                          │
│  牛乳を買う|              │  ← テキスト入力欄
│                          │
│                          │
│                          │
│                          │
│                          │
│                          │
│                          │
│                          │
│                          │
└─────────────────────────┘
```

### ヘッダー

| 要素 | 仕様 |
|------|------|
| 戻るボタン | 左端。タップで一覧画面へ戻る |
| 文字数表示 | 右端に `現在の文字数/100` を表示 |
| 文字数の色 | 80文字以上で警告色（オレンジ）、100文字で赤 |

### テキスト入力

| 要素 | 仕様 |
|------|------|
| 入力欄 | 画面全体を使った大きな入力エリア |
| フォント | 選択中モードの本文フォント |
| フォントサイズ | 18pt〜24pt（モードによって異なる） |
| プレースホルダー | 「メモを入力...」（モードのフォントで表示） |
| キーボード | テキスト入力画面遷移時に自動でキーボード表示 |
| returnキー | `done`（キーボードを閉じる） |
| 最大文字数 | 100文字（超過分は入力不可） |
| 自動保存 | 入力変更から500msのデバウンス後にAsyncStorageへ保存 |

### 背景

| 要素 | 仕様 |
|------|------|
| テクスチャ | モードに応じた背景テクスチャを表示 |
| Fountain Pen | クリーム色ペーパーテクスチャ |
| Typewriter | 古い紙のテクスチャ |
| Modern Ink | マットブラック（テクスチャなし） |

## 4. 画面遷移図

```
[アプリ起動]
    │
    ├── 初回起動 ──→ [オンボーディング] ──→ [メモ一覧]
    │
    └── 2回目以降 ──→ [メモ一覧]
                          │
                          ├── FAB(＋)タップ ──→ [メモ編集（新規）]
                          │                         │
                          │                         └── 戻る ──→ [メモ一覧]
                          │
                          └── メモタップ ──→ [メモ編集（既存）]
                                                │
                                                └── 戻る ──→ [メモ一覧]
```

### 遷移アニメーション

| 遷移 | アニメーション |
|------|--------------|
| 一覧 → 編集 | 右からスライドイン（iOS標準） |
| 編集 → 一覧 | 左へスライドアウト（iOS標準） |
| オンボーディング → 一覧 | フェードアウト/フェードイン |
