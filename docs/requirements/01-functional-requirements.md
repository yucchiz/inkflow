# 01 - 機能要件

## 1. メモ（TODO）機能

### 1.1 メモの作成

| 項目 | 仕様 |
|------|------|
| 作成方法 | 一覧画面右下のFAB（＋ボタン）をタップ → 編集画面へ遷移 |
| テキスト形式 | 単一行（改行不可） |
| 最大文字数 | 100文字 |
| 初期状態 | テキスト空、未完了（`completed: false`） |
| ID生成 | UUID v4（`crypto.randomUUID()`） |
| 保存方式 | 自動保存（テキスト変更のたびにAsyncStorageへ保存） |
| 空メモの扱い | テキスト未入力のまま編集画面を離れた場合、メモは自動削除される |

### 1.2 メモの編集

| 項目 | 仕様 |
|------|------|
| 遷移方法 | 一覧画面でメモをタップ → 編集画面へ遷移 |
| 保存タイミング | テキスト変更のたびに自動保存（デバウンス: 500ms） |
| 戻る操作 | ヘッダーの戻るボタン、またはスワイプバックで一覧へ戻る |
| 文字数カウント | 入力欄付近に残り文字数を表示（100文字に近づいたら警告色） |

### 1.3 メモの削除

| 項目 | 仕様 |
|------|------|
| 操作方法 | 一覧画面でメモアイテムを左スワイプ → 即削除 |
| Undo | 削除後、画面下部に「元に戻す」トーストを3秒間表示 |
| Undoの挙動 | トーストタップで削除前の位置にメモを復元 |
| トースト消滅後 | AsyncStorageから完全に削除。復元不可 |

### 1.4 メモの完了/未完了切替

| 項目 | 仕様 |
|------|------|
| 操作方法 | 一覧画面でチェックボックスをタップ |
| 完了時の表示 | テキストに取り消し線を表示。フォントに合わせたスタイル適用 |
| リスト内の位置 | 完了しても位置は変わらない（そのままの位置を維持） |
| 未完了への戻し | チェックボックスを再度タップで未完了に戻る |

### 1.5 メモの並び替え

| 項目 | 仕様 |
|------|------|
| 操作方法 | メモアイテムを長押し → ドラッグ&ドロップで並び替え |
| ドラッグ開始 | 長押し（約300ms）でドラッグモードに入る |
| ドラッグ中の表示 | 持ち上げたアイテムが浮き上がり、影がつく |
| ドロップ時 | 各メモの `order` フィールドを更新し、AsyncStorageに保存 |
| ライブラリ | react-native-draggable-flatlist |

### 1.6 メモの上限

| 項目 | 仕様 |
|------|------|
| 最大保存件数 | 50件 |
| 上限到達時 | FABボタンを非活性化し、「メモは最大50件です」のメッセージを表示 |

## 2. タイポグラフィモード

### 2.1 モード一覧

| モード名 | フォント（本文） | フォント（見出し） | 背景 | 特殊効果 |
|----------|-----------------|-------------------|------|---------|
| Fountain Pen | Lora (Regular) | Playfair Display (Bold) | クリーム色ペーパーテクスチャ | インク滲みアニメーション |
| Typewriter | Courier Prime (Regular) | Special Elite (Regular) | 古い紙のテクスチャ | 文字のブレ、キャレットアニメーション |
| Modern Ink | Cormorant Garamond (Regular) | DM Serif Display (Regular) | マットブラック | 文字フェードイン、光沢感 |

### 2.2 モード切替

| 項目 | 仕様 |
|------|------|
| 切替UI | 一覧画面ヘッダーに3つのアイコンを表示 |
| 切替操作 | アイコンタップで即座にモード切替 |
| 切替アニメーション | カラー・フォントがスムーズにトランジション |
| デフォルトモード | Modern Ink（初回起動時） |
| 設定の永続化 | モード切替時にAsyncStorageへ即座に保存 |

## 3. 設定

### 3.1 設定項目

| 設定名 | デフォルト値 | 説明 |
|--------|------------|------|
| typographyMode | `'modern-ink'` | 選択中のタイポグラフィモード |
| soundEnabled | `false` | タイプライター音（Phase 2） |
| hapticsEnabled | `false` | ハプティクス（振動フィードバック） |

### 3.2 ハプティクス

| 項目 | 仕様 |
|------|------|
| デフォルト | OFF |
| 対象操作 | チェック切替、削除、モード切替、ドラッグ開始/終了 |
| 設定変更 | 将来的に設定画面で切替可能にする（MVPではコード内定数で管理） |

---

## 4. オンボーディング

| 項目 | 仕様 |
|------|------|
| 表示タイミング | 初回起動時のみ |
| 画面数 | 1〜2画面 |
| 内容 | 3つのタイポグラフィモードの紹介 |
| スキップ | スキップボタンで即座にメモ一覧へ遷移可能 |
| 再表示 | 二度と表示されない（AsyncStorageでフラグ管理） |
