# 05 - アニメーション仕様

## 概要

全アニメーションは `react-native-reanimated` で実装する。
ジェスチャー操作は `react-native-gesture-handler` を使用する。

---

## 1. リスト表示アニメーション（スタガーアニメーション）

メモ一覧画面の初回表示時、メモアイテムが上から順番に登場する。

| パラメータ | 値 |
|-----------|------|
| アニメーション種別 | フェードイン + 下方向からスライド |
| 各アイテムの遅延 | 50ms × index |
| duration | 300ms |
| イージング | `Easing.out(Easing.cubic)` |
| 初期位置 | translateY: 20, opacity: 0 |
| 最終位置 | translateY: 0, opacity: 1 |
| トリガー | 画面マウント時、モード切替時 |

---

## 2. チェックボックスアニメーション

メモの完了/未完了切替時に満足感のあるフィードバックを提供する。

### 未完了 → 完了

| パラメータ | 値 |
|-----------|------|
| チェックマーク | スケールイン（0 → 1）+ 回転（-10deg → 0deg） |
| duration | 250ms |
| イージング | スプリング（damping: 12, stiffness: 180） |
| テキスト取り消し線 | 左から右へ線が伸びるアニメーション（200ms） |
| テキスト色 | アクセントカラー → completedカラーへフェード（200ms） |

### 完了 → 未完了

| パラメータ | 値 |
|-----------|------|
| チェックマーク | フェードアウト（opacity: 1 → 0, 150ms） |
| 取り消し線 | 即座に消える |
| テキスト色 | completedカラー → textカラーへフェード（150ms） |

---

## 3. スワイプ削除アニメーション

左スワイプでメモを削除する際のアニメーション。

| パラメータ | 値 |
|-----------|------|
| スワイプ方向 | 左方向のみ |
| 削除閾値 | 画面幅の30%以上スワイプで削除確定 |
| 背景色 | スワイプ中に赤背景 + ゴミ箱アイコンが露出 |
| 削除確定時 | アイテムが左にスライドアウト + 高さが0に縮小（200ms） |
| イージング | `Easing.inOut(Easing.ease)` |

---

## 4. ドラッグ&ドロップアニメーション

長押しでメモアイテムをドラッグして並び替えする。

| パラメータ | 値 |
|-----------|------|
| ドラッグ開始 | 長押し300ms後にアクティブ化 |
| 持ち上げ効果 | scale: 1.03, shadowOpacity: 0.2, elevation: 5 |
| ドラッグ中 | 他のアイテムがスムーズに上下にスライドして場所を空ける |
| ドロップ | スプリングアニメーションで目標位置に着地 |
| スプリング設定 | damping: 20, stiffness: 200 |
| ライブラリ | react-native-draggable-flatlist |

---

## 5. 画面遷移アニメーション

| 遷移 | 種別 | 詳細 |
|------|------|------|
| 一覧 → 編集 | スライド | 右からスライドイン（Expo Routerのデフォルト） |
| 編集 → 一覧 | スライド | 左へスライドアウト（スワイプバック対応） |
| オンボーディング → 一覧 | フェード | opacity遷移（duration: 400ms） |

---

## 6. モード切替アニメーション

タイポグラフィモードを切り替えた際の全体的なトランジション。

| パラメータ | 値 |
|-----------|------|
| 背景色 | 現在のカラー → 新モードのカラーへ補間（300ms） |
| テキスト色 | 同上 |
| フォント | 即座に切替（フォントのクロスフェードは技術的に困難） |
| ヘッダーアイコン | 選択アイコンのアクセントカラーがフェード遷移（200ms） |
| イージング | `Easing.inOut(Easing.ease)` |

---

## 7. FABアニメーション

| パラメータ | 値 |
|-----------|------|
| 表示 | 画面マウント時にスケールイン（0 → 1, 300ms, スプリング） |
| タップ | スケールダウン → スケールアップ（0.9 → 1.0, 100ms） |
| 非活性時 | opacity: 0.4、タップアニメーションなし |

---

## 8. Undoトーストアニメーション

| パラメータ | 値 |
|-----------|------|
| 表示 | 下からスライドイン（translateY: 100 → 0, 200ms） |
| 消滅 | フェードアウト（opacity: 1 → 0, 200ms） |
| タイムアウト | 3秒後に自動で消滅アニメーション開始 |
| イージング | 表示: `Easing.out(Easing.cubic)` / 消滅: `Easing.in(Easing.ease)` |

---

## 9. モード別特殊エフェクト

### 9.1 Fountain Pen — インク滲みエフェクト

メモ編集画面でテキスト入力時に発動する。

| パラメータ | 値 |
|-----------|------|
| トリガー | 文字入力のたび |
| 効果 | 入力文字の周囲に微かなインク滲み（opacity: 0.15のブラー円） |
| サイズ | 文字サイズの1.5倍の円 |
| duration | 滲みが広がる: 200ms → フェードアウト: 400ms |
| カラー | アクセントカラー（#2C3E6B）を半透明で使用 |

### 9.2 Typewriter — 文字ブレ + キャレットアニメーション

| パラメータ | 値 |
|-----------|------|
| 文字ブレ | テキスト全体にごくわずかなランダムオフセット（±0.3px） |
| ブレの更新 | レンダリング時に一度だけ計算（入力中は動かない） |
| キャレット | ブリンクアニメーション（点滅間隔: 530ms） |
| キャレットスタイル | 太めのブロックキャレット（Typewriterモード専用） |

### 9.3 Modern Ink — フェードイン + 光沢感

| パラメータ | 値 |
|-----------|------|
| テキストフェードイン | 編集画面遷移時にテキストがフェードイン（opacity: 0 → 1, 400ms） |
| 光沢感 | テキストに微かなグラデーションオーバーレイ（ゴールドアクセント、opacity: 0.05） |
| イージング | `Easing.out(Easing.cubic)` |
